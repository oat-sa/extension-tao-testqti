MNOP Stress Test Suite

This test suite evaluates MNOP calculation performance under various stress conditions:

  • 50 items - Basic performance baseline
  • 100 items - With categories and weights
  • 500 items - Large test stress
  • 1000 items - Extreme stress test
  • Deep nesting - Nested section performance
  • Sequential calculations - Cache efficiency
  • Category aggregation - Multi-category stress
  • Memory cycles - Init/clear cycle efficiency
  • Selection rules - Complex rule handling
  • Real-world scenario - 250 items with full complexity

Performance Targets:

  • 50 items: <100ms
  • 100 items: <200ms
  • 500 items: <500ms
  • 1000 items: <1000ms

Check browser console for detailed timing information.
